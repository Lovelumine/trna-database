# tRNAtherapeutics Curation Scripts

Scripts used to build the GitHub-safe curation reports and local evidence workspace for `Engineered_sup_tRNA`.

The scripts expect the local ignored workspace `field-curation-workdir/` and local database environment files to exist. They should not be assumed to run in GitHub CI without local MySQL, downloaded source files, and AF3/LLM-tRNAAlign dependencies.
